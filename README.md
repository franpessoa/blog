# sietch.link: um blog
Features: 
 - KaTeX support
 - bilingual and accessible
 - tags system
 - uses Astro as a SSG

 ## Development
 This project uses PNPM and is deployed via Cloudflare Wrangle.
 
 ```sh
 pnpm install # install dependencies
 pnpm dev # dev server at http://localhost:4321
 pnpm build # build for production at ./dist
 ```
